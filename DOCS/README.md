# 📚 DOCS - 项目文档目录

本目录包含viiyd3.0项目的所有规划、标准和协议文档。

## 📋 文档列表

### 业务规划
- **BUSINESS_PLAN.md** - 业务计划，包含项目商业目标和策略
- **ROADMAP.md** - 项目路线图，功能开发时间线

### 技术标准
- **STANDARDS.md** - 项目技术标准和编码规范
- **DESIGN_SYSTEM.md** - 设计系统规范（颜色、字体、组件等）
- **WORK_TEMPLATE.md** - Work内容文章标准模板
- **WORK_REFACTOR_PROTOCOL.md** - Work内容重构协议

### 管理流程
- **MANAGEMENT_PLAN.md** - 项目管理计划和流程
- **HANDOVER_PROTOCOL.md** - 项目交接协议
- **MIGRATION_PLAN.md** - 数据迁移计划
- **implementation_plan.md** - 当前正在执行的实施计划

## 📖 使用指南

### 新建Work文章
1. 参考 `WORK_TEMPLATE.md` 了解标准格式
2. 查阅 `STANDARDS.md` 确保符合项目规范
3. 查看 `DESIGN_SYSTEM.md` 使用正确的设计元素

### 项目交接  
1. 阅读 `HANDOVER_PROTOCOL.md` 了解交接流程
2. 参考 `MANAGEMENT_PLAN.md` 理解项目管理结构

### 内容重构
1. 遵循 `WORK_REFACTOR_PROTOCOL.md` 的重构步骤
2. 参考 `MIGRATION_PLAN.md` 处理数据迁移

## 🔄 文档更新

所有文档应保持最新：
- 技术变更时更新 STANDARDS.md
- 设计变更时更新 DESIGN_SYSTEM.md  
- 模板变更时更新 WORK_TEMPLATE.md
- 流程变更时更新对应的管理文档
